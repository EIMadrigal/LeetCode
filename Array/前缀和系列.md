## 560 和为 K 的子数组
```java
// O(n^2), O(1)
class Solution {
    // 枚举以i为起点的所有子数组
    public int subarraySum(int[] nums, int k) {
        int len = nums.length;
        int ans = 0;
        for (int i = 0; i < len; ++i) {
            int sum = 0;
            for (int j = i; j < len; ++j) {
                sum += nums[j];
                if (sum == k) {
                    ++ans;
                }
            }
        }
        return ans;
    }
}


// O(n), O(n)
class Solution {
    // 枚举以i为终点的子数组判断是否满足条件: sum(j...i) == k, 0 <= j <= i
	// 又sum(0...i)已知, 故只需判断sum(0...j) == sum(0...i) - k出现的次数, 即为j的所有可能的取值次数
    public int subarraySum(int[] nums, int k) {
        // hashmap: presum -> #occurence
        HashMap<Integer, Integer> cnt = new HashMap<>();
        int len = nums.length, sum = 0, ans = 0;
        cnt.put(0, 1);
        for (int i = 0; i < len; ++i) {
            sum += nums[i];
            // if previous pos has sum - k, then sum - (sum - k) = k
            if (cnt.containsKey(sum - k)) {
                ans += cnt.get(sum - k);   
            }
            cnt.put(sum, 1 + cnt.getOrDefault(sum, 0));
        }
        return ans;
    }
}
```
